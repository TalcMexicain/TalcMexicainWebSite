AudioJDR

Une application accessible pour les jeux de rôle textuels

Description du Projet

AudioJDR est une application Windows innovante conçue pour rendre les jeux de rôle textuels accessibles aux personnes déficientes visuelles. Grâce à la synthèse et à la reconnaissance vocale, les joueurs peuvent interagir avec le jeu uniquement par la voix, sans nécessiter d'interface graphique.

Fonctionnalités Principales

Partie Joueur

  • Synthèse vocale pour la lecture des descriptions
  • Reconnaissance vocale pour les interactions
  • Sauvegarde automatique des parties
  • Support multilingue

Partie Éditeur

  • Création de jeux audio interactifs
  • Interface intuitive de conception
  • Import/Export au format JSON
  • Gestion avancée des événements
